\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{🎮 Features \& Controls}{🎮 Features \& Controls}}\label{index_autotoc_md2}
The emulator uses SFML 3.\+0 to render a robust UI in both standard mode and an advanced AMOLED Debug View. Both modes fully support the following features:\+


\begin{DoxyItemize}
\item {\bfseries{Turbo Mode (2x Speed)}}:\+ Toggle with {\ttfamily T}
\item {\bfseries{Pause/\+\+Play}}:\+ Toggle with {\ttfamily Space}
\item {\bfseries{Step Instruction}}:\+ Press {\ttfamily N} (while paused)
\item {\bfseries{Hard Reset}}:\+ Press {\ttfamily R}
\item {\bfseries{Save/\+\+Load States (Internal Slots)}}:\+ Press {\ttfamily 0-\/9} to Load, {\ttfamily Shift + 0-\/9} to Save.
\item {\bfseries{Save/\+\+Load States (External File)}}:\+ Press {\ttfamily E} to Load, {\ttfamily Shift + E} to Save (via Zenity File Picker).
\item {\bfseries{Gamepad Control}}:\+
\begin{DoxyItemize}
\item {\ttfamily Arrow Keys} for D-\/\+Pad
\item {\ttfamily Z} for A
\item {\ttfamily X} for B
\item {\ttfamily Enter} for Start
\item {\ttfamily Right Shift} for Select
\end{DoxyItemize}
\end{DoxyItemize}

{\itshape Note:\+ In Normal mode, system hotkey triggers will display a temporary bright-\/red confirmation overlay.}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{🖥️ Display Modes}{🖥️ Display Modes}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Standard Mode}}:\+ Plays the ROM with a 1.\+0x native display.
\item {\bfseries{Debug Mode ({\ttfamily -\/-\/debug})}}:\+ Opens a stunning 16:\+9 (1600x900) AMOLED-\/themed developer UI featuring:\+
\begin{DoxyItemize}
\item 5x Upscaled Game Screen
\item Live CPU Registers (AF, BC, DE, HL, SP, PC, and Flags)
\item Real-\/time Opcode Trace
\item Clickable virtual gamepad
\item Memory Read/\+\+Write metrics
\item Live ROM hex-\/byte monitoring
\end{DoxyItemize}
\item {\bfseries{Fullscreen ({\ttfamily -\/-\/fullscreen})}}:\+ Upscales both Normal and Debug modes to 1920x1080 without distorting the pixel aspect ratio.
