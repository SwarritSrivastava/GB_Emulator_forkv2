{\bfseries{Game Boy (DMG-\/01) Emulator}} ‚Äì COPS 26 Project

Early-\/stage emulator for the original Game Boy (Dot-\/\+Matrix Game ‚Äì DMG-\/01).

Currently focused on {\bfseries{accurate CPU core}} and {\bfseries{basic \doxylink{classMMU}{MMU} /\+ cartridge handling}}.

 ~\newline
 {\itshape (classic gray brick ‚Äì the real hardware this project aims to emulate)}\hypertarget{md_README_autotoc_md831}{}\doxysection{\texorpdfstring{Current Status (March 2026)}{Current Status (March 2026)}}\label{md_README_autotoc_md831}
{\bfseries{What works reasonably well:\+}}


\begin{DoxyItemize}
\item Z80-\/like CPU core (Sharp LR35902)
\begin{DoxyItemize}
\item Most 8-\/bit \& 16-\/bit instructions implemented (via {\ttfamily opcodes.\+cpp} + {\ttfamily cb\+\_\+opcodes.\+cpp})
\item CB-\/prefixed opcodes table
\item Correct reset state (AF=01B0h, BC=0013h, DE=00D8h, HL=014Dh, SP=FFFEh, PC=0100h)
\item Basic cycle-\/accurate stepping (PC increments, NOPs, HALT behavior)
\item Unknown opcode handling (graceful fallback)
\end{DoxyItemize}
\item \doxylink{classMMU}{MMU} /\+ Memory Map
\begin{DoxyItemize}
\item ROM loading \& mapping (fixed bank 0 + switchable banks not yet)
\item Cartridge header parsing (title, MBC type, ROM/\+\+RAM size, checksum)
\item WRAM (C000‚Äì\+DFFF), basic echo RAM behavior
\item Read-\/only ROM protection
\item Zero-\/initialized memory on startup
\end{DoxyItemize}
\item Very basic execution loop
\begin{DoxyItemize}
\item Loads \& runs ROM until stopped (currently limited instructions executed)
\item Prints CPU state after reset
\end{DoxyItemize}
\end{DoxyItemize}

{\bfseries{What is NOT implemented yet:\+}}


\begin{DoxyItemize}
\item PPU /\+ rendering (no screen output)
\item Timer, DMA, interrupts (IME, vblank, joypad, etc.)
\item Sound
\item Input (joypad)
\item MBC1/\+\+MBC3/\+... bank switching (only basic ROM mapping)
\item Proper frame timing \& synchronization
\item Save RAM (external cartridge RAM)
\item Display window /\+ SDL /\+ graphics backend
\end{DoxyItemize}

\begin{DoxyItemize}

\item Tetris 2 (executes \texorpdfstring{$\sim$}{\string~}140 instructions after reset)
